# OpenTK Lighting
 
 Loaders are scripts to help load mesh or image data into this project.

 Shaders are formated like {
 
	Shader:
		ShaderName:
			vertex.glsl
			fragment.glsl
 }

 Objects (File for object data) are formated like {
	
	Objects:
		ObjectName:
			Textures:
				color.png
				normal.png
				specular.png
 }
